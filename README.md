# Mping

## Professional Network Monitoring for Live Event Production

Mping is a native macOS SwiftUI application for monitoring and
visualising complex AV networks used in live production.

## Current Features

-   Real-time ICMP monitoring
-   Offline verification engine
-   LLDP topology discovery
-   SNMP telemetry
-   Fibre link visualisation
-   Interactive topology canvas
-   Device inspector
-   Alerting framework

## Supported Hardware

-   L-Acoustics
-   Netgear AV Line
-   Generic SNMP devices
-   Fibre infrastructure

## Philosophy

-   Reliability before features
-   Minimal false alerts
-   Small iterative development
-   Engine/UI separation where practical
-   Future Windows portability
